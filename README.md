# Birthday Paradox

## Descripción

🖥 Simulación que compara las coincidencias de cumpleaños entre n mapaches, en donde, se usa como entrada n (Cantidad de mapaches). Prueba la veracidad de la llamada "Paradoja del Cumpleaños".
